# Decidim::Slider

A Slider component for Decidim's homepage.

## Usage

Slider will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-slider"
```

And then execute:

```bash
bundle
```
